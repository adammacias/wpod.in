# Guia

# O que é o Odin?

Odin é um framework desenvolvido pela comunidade @wpbrasil para chutar bundas auxiliar o desenvolvimento de temas para WordPress.

# Download

Baixar versão 2.x .zip

# Instalação

Envie a pasta "odin" para o diretório de temas padrão do WordPress wp-content/themes ou simplesmente envie o arquivo zipado utilizando o instalador de temas do WordPress.

# Destaques
